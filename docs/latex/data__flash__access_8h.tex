12.\+2.\+45 Data Flash Access() 0x4000–0x5\+FFF headers 

Accessing data flash (DF) is only supported by the \mbox{\hyperlink{alt__manufacturer__access_8cpp_ab30f45d45249d982ae8e5fc89403c400}{Alt\+Manufacturer\+Access()}} by addressing the physical address.

\begin{DoxyWarning}{Warning}
\mbox{[}!\mbox{]} Not Available in SEALED Mode. UNSEALED Mode is enough for both\+: reading and writing. If you try to read the DF in SEALED mode, you\textquotesingle{}ll just get an array of zeros.
\end{DoxyWarning}

The Data length should be in the range \mbox{[}1; 32\mbox{]}
