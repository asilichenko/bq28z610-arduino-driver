The checksum is the bitwise inversion of the sum of bytes.

For example\+: 
\begin{DoxyPre}
  \string~(0x35 + 0x00 + 0x23 + 0x01 + 0x67 + 0x45 + 0xAB + 0x89 + 0xEF + 0xCD)
  = \string~(0xF5) = \string~(0b11110101) = 0b00001010 = 0x0A
\end{DoxyPre}
